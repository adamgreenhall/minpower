Solver Choices¶

GLPK¶

  • free

  • open source

  • easy to install

  • slow for large problems

CBC¶

  • free

  • open source

  • faster than GLPK for large problems

Gurobi¶

  • free with academic license

  • easy to install

  • usually ranks as the fastest available MIP solver

CPLEX¶

  • free with IBM academic license

  • difficult to install

  • industry standard for large problems
